# **FUNNEL ANALYSIS**
Welcome to my personal project!

This repository contains my mini project using Funnel Analysis framework for reporting.

## 📊 **Product Performance Analysis & Conversion Rate Optimization**
An analysis utilizing customer journey data to identify top-performing items and locate bottlenecks in the purchasing process.
* **Revenue & Return Monitoring:** Identified "The Original Mr. Fuzzy" as the top net revenue generator. Conversely, flagged "The Birthday Sugar Panda" for having the highest return rate, highlighting the need to review manufacturing processes or website product descriptions.
* **Funnel Bottleneck Identification:** Discovered a severe merchandising issue with "The Forever Love Bear". While its conversion from "View Detail" to "Add-to-Cart" is exceptionally strong (58% on Desktop, 51% on Mobile), its initial Click-Through Rate (CTR) from the product catalog was only 10%, compared to over 60% for "Mr. Fuzzy". Proposed immediate A/B testing for product thumbnails to increase visual appeal.
* **Device-Specific UI/UX Friction:** Identified critical drop-offs occurring specifically on mobile devices. The CTR to view product details for "Mr. Fuzzy" was 66% on Desktop but dropped to 51% on Mobile. Recommended immediate end-to-end user testing to fix mobile checkout flows and optimize the mobile product listing layout.

🔧 Tech Stack
* **Pandas & NumPy –** Data merging, handling missing values, aggregation, and funnel step metric calculations.
* **Plotly –** Interactive data visualization, specifically utilizing `go.Funnel` and `make_subplots` to compare conversion funnels across Desktop and Mobile devices. 
* **Matplotlib & Seaborn –** Static data visualization for revenue distribution, refund rates, and step-to-step CTR comparisons.
